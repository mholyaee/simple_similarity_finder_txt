3_python_bidi = command = pip install python-bidi

how it works :
this program need 2 file in format of txt to check similarity , it check each parageraph with other paragheraphs of second file ,
and and find words that they are same between parageraphs and more than 3 length and repeated more than 2 in file.
at last it return the Average of all data ( data is the number that show the similarity between to parageraph).
